Brainium
An excellent tool for Forecasting and Managing Inventory

This module is an easy interactive tool to see all the inventory analysis in one place. Using easy to use web based UI you can simply specify the seasonality and buffer for different product categories month wise and then see the calculations reflecting for 6 months in future to give you a picture of how the inventory would look like in multiple locations/warehouse.

This module is in particular beneficial to large sellers who have numerous product to manage, plan in advance for manufacturing and shipping to have the quantities replenished the warehouses. The current module is designed more specifically for Amazon/online sellers who sell in multiple marketplaces and need an easy historical and forecast view of their products. With the ability track logs, the application can keep a record of all the activities done by your employees and thus, those who are using excel for their calculations can now be relieved of managing complex formulas and macros and being fearful about the employees making mistake and jeopardizng the excel sheet. If you need any help feel free to get in touch with us on email info@techneith.com or on Skype at Odoosetup

The salient features of the module are

React based WEb interface - Easy React based interactive interface which works smooth and fast similar to excel.

3 way Table view - A wholistic view of historical weekly sales, current inventory and forecast view all in one place

Easy formula management for forecasting - An easy way to change the seasonality and buffer and see the numbers dancing yourself to better evaluate stock size at beginning of every month

Make Secure logging and Root Cause Analysis - Since all activities are recorded and tracked you can be less bothered about employees making mistake and not being able to find the root cause.

Simply open the saved graphs to see the query re-computed on live data.

Excel like interface - Already habituated with excel? Well we have kept that in mind and given a similar view for lesser context switching for your team.

Inventory and Purchase upload - Not on Odoo fully? There is special section to upload inventory and purchase separately if you dont want the forecasting calculation to depend on inventory value from Odoo.

Marketplaces page

After login see all the marketplaces that the particular user has access to

Settings page

Easily define the manufacturing time, air and sea shipping time for the particular marketplace and and set the buffer and seasonality mutiplier for different product categories

Inventory and Purchase upload

Do not want to reply on current stock value in Odoo? Easily upload inventory and purchase orders as excel for different marketplaces to refine your calculations

Logs view

Out of sync SKUs, categories, new products which have undefined properties, etc will all be shown here so that you can take proper action on those.
